
# 🌦 Weather System Dashboard

A modern Weather Dashboard Web Application that allows users to search any city and view real-time weather information.

The application fetches live weather data using API integration and displays it in a clean and interactive user interface.

---

## 🚀 Features

✔ Search weather by city name  
✔ Real-time temperature display  
✔ Shows humidity level  
✔ Displays wind speed  
✔ Weather condition icons  
✔ Loading animation while fetching data  
✔ Error handling for invalid city names  
✔ Responsive modern UI

---
